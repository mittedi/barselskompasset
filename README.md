# Barselskompasset

Et gratis, vejledende beregnerværktøj til planlægning af barselsorlov for funktionærer i Danmark.

## Om projektet

Barselskompasset hjælper dig med at få overblik over din barselsorlov baseret på:
- **Barselsloven** (LBK nr. 816 af 03/06/2021 med ændringer)
- **Funktionærloven** §7

Værktøjet er en enkelt HTML-fil — ingen installation, ingen afhængigheder, ingen server. Åbn bare `index.html` i en browser.

## Funktioner

- Beregner for mor, far og medforældre
- Trin-for-trin guide
- Tilpassede perioder (faste lovpligtige + variable efter arbejdsgivers politik)
- Print til PDF
- Fungerer offline

## Vigtig note

Beregningerne er **vejledende**. Lønnede perioder ud over det lovpligtige afhænger af din individuelle ansættelseskontrakt eller personalepolitik. Kontakt din arbejdsgiver eller HR ved tvivl.

## Kom i gang

1. Download eller klon repoet
2. Åbn `index.html` i en browser
3. Følg de 4 trin
